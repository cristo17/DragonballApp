# Dragon Ball Codex 🐉

**Dragon Ball Codex** es una enciclopedia digital completa diseñada para los fanáticos de la legendaria serie de Akira Toriyama. Esta aplicación permite explorar una base de datos de **100 guerreros**, desde los clásicos de Dragon Ball Z hasta los más recientes de Dragon Ball Super y las películas.

Desarrollado por **Cristo17**.

## ✨ Características Principales

- **100 Guerreros Registrados**: Una combinación precisa entre la API oficial y personajes legendarios añadidos manualmente (SSJ4, Villanos de películas, Torneo del Poder, etc.).
- **Ficha Técnica Detallada**: Cada personaje cuenta con 11 puntos de información clave:
  - Nombre, Raza, Género, Afiliación y Planeta de Origen.
  - Niveles de Poder: Ki Base y Ki Máximo.
  - Historia/Biografía completa.
  - Técnicas Especiales (Kamehameha, Final Flash, etc.).
- **Diseño Premium**: Interfaz moderna utilizando Material 3, con gradientes temáticos, tarjetas elevadas y una paleta de colores inspirada en las Esferas del Dragón.
- **Splash Screen Animada**: Una entrada épica con la Esfera del Dragón y efectos de aparición suave.
- **Totalmente en Español**: Todas las etiquetas, razas y descripciones técnicas han sido traducidas para una mejor experiencia.

## 🛠️ Tecnologías Utilizadas

- **Lenguaje**: Kotlin
- **Interfaz**: Jetpack Compose (Moderno y Declarativo)
- **Networking**: Retrofit & Gson (Conexión a Dragon Ball API)
- **Imágenes**: Coil (Carga asíncrona de artes oficiales)
- **Navegación**: Navigation Compose
- **Arquitectura**: Arquitectura Limpia (Data, Domain, UI)

## 📸 Previsualización

La aplicación cuenta con una lista fluida de 100 personajes y una pantalla de detalle robusta que resalta las estadísticas de poder de cada guerrero.
